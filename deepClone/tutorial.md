# 深拷贝

## 理解

+ 简单理解：b是a的一份拷贝，b中没有对a中对象的引用；
+ 另一种理解：b是a的一份拷贝，把b和a画成图，a和b之间没有连结；

## 如何答题

1. 询问数据类型
2. 询问数据规模
3. 询问性能要求
4. 询问运行环境
5. 询问其他要求
6. 开始写

## 方法

### JSON序列化反序列化

```js
var a0 = xxx;
var a1 = JSON.parse(JSON.stringify(a1));
```

缺点：

1. 不支持function；
2. 不支持undefined；
3. 不支持引用：`a.self = a`；
4. 不支持Date：会把Date变成日期字符串；
5. 不支持正则；
